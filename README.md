# 🌊 Oil Spill Detection using Satellite Imagery with Deep Learning

## 📌 Overview

Oil spills in oceans are a major environmental threat that affects marine life and coastal ecosystems. This project uses Deep Learning and Computer Vision to automatically detect oil spills from satellite images and notify the concerned authorities through email alerts.

---

## 🎯 Objectives

- Detect oil spills using satellite images.
- Classify images as Oil Spill or No Oil Spill.
- Display prediction results through a Flask web application.
- Send email alerts to coastal authorities when an oil spill is detected.

---

## 🚀 Features

- User Login System

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/prago-dev/oil-spill-detection.git
```

### Go to Project Folder

```bash
cd oil-spill-detection
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Project

```bash
python main.py
```
